# Trading System - Automated Trading & Analysis

## Overview
Integrated trading system with 5 core modules for automated performance tracking, real-time alerts, weekly reports, contrarian indicators, and Monte Carlo simulations.

## Modules
1. Performance Tracker
2. Alert System  
3. Lessons Learned Reports
4. Contrarian Dashboard
5. Monte Carlo Portfolio Simulator

## Assets Tracked
- Gold
- Silver
- Bitcoin
- S&P 500
- Brent Oil
